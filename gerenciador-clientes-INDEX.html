🔥 Configurar Firebase
Passo 1 — Criar projeto no Firebase
Acesse console.firebase.google.com → Clique em Adicionar projeto → Dê um nome (ex: gr7-instalacoes) → Desative Google Analytics → Criar projeto.
Passo 2 — Criar banco Realtime Database
No menu lateral clique em Build → Realtime Database → Criar banco de dados → Escolha a região us-central1 → No modo de segurança, selecione Modo de teste → Ativar.
Passo 3 — Registrar App Web
No menu lateral, clique em ⚙️ Configurações do projeto → Role até Seus apps → Clique em </> (Web) → Dê um nome → Registrar app. Copie os dados do firebaseConfig e cole abaixo.
Passo 4 — Colar configuração abaixo e salvar
As configurações ficam salvas neste navegador. Cada pessoa da equipe precisa fazer isso uma vez.
API Key
Auth Domain
